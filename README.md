# Quill

Quill is a journal taking web application for a digital bullet journal app. The website presents a calm journaling product.

## Live Demo

[View the live site]( https://charle-ux.github.io/quill/)

## Overview

The project is built as a static website using HTML and CSS for now. It is focused on layout practice, responsive design, reusable styling, and a polished web-page experience for a journaling product.

## Features

- Responsive page layout
- Navigation with logo, login, and register links
- Hero section with primary and secondary call-to-action buttons
- Feature section for journaling benefits
- Two-column desktop feature layout
- Single-column mobile feature layout
- Call-to-action card
- Footer with social media links
- Floating chat icon
- Extra practice pages for CSS Grid, Flexbox, lists, paragraphs, login, registration, and journal creation

## Tech Stack

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Media Queries
- Google Fonts

## Main Files

- `quill/INDEX.HTML` - Main landing page
- `quill/app.css` - Main stylesheet for typography, colors, layout, buttons, hero, features, CTA, footer, and chat icon
- `quill/queries.css` - Responsive styles for tablet and mobile screens
- `quill/login.html` - Login placeholder page
- `quill/REGISTER.HTML` - Register placeholder page

## Responsive Design

The website uses `queries.css` to adjust the layout across screen sizes. On smaller screens, the feature grid changes from two columns to one column, the hero spacing is reduced, and the CTA card becomes more compact.

## Status

This is a front-end static website. The login, register, and chat pages are currently placeholders or practice pages and do not include backend functionality at this point of the Project.
This is simply a Minimum viable Product
